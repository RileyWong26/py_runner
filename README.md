# Py Runner
A CLI tool to run python files. Py Runner opens a list containing directories of python files. The user can navigate using the list and select a python file to run it, saving time on remembering character sensitive python file names to run.

## How to Run

### Quick Run
```
cargo run 
```

### Run as Binary
```
# Build in a binary
cargo build --release

# Run the binary
./target/release/py_runner   
```
